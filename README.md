# 💬 NebulaChat & MeetFlow Projects

This repository showcases two real-time web applications — **NebulaChat** and **MeetFlow** — demonstrating advanced full-stack development with real-time communication, privacy-focused design, and responsive user interfaces.

---

## 🚀 Projects Overview

### 🟣 **NebulaChat – Anonymous Real-Time Chat Platform (Oct 2025)**

**Description:**  
Built a privacy-focused chat app with Next.js and Socket.IO supporting real-time messaging without any data storage. Designed for instant, anonymous communication with a modern and responsive UI.

**✨ Features**
- Anonymous login without authentication
- Real-time instant messaging via Socket.IO
- Ephemeral chats (no database or message storage)
- Responsive and mobile-friendly UI
- Smooth animations with Framer Motion

**🧰 Tech Stack**
- **Frontend:** Next.js, React.js, Tailwind CSS, Framer Motion  
- **Backend:** Node.js, Express.js, Socket.IO  
- **Other Tools:** ESLint, PostCSS, Render (deployment)  

**🌐 Live Demo:** [nebulachat.onrender.com](https://nebulachat.onrender.com)
